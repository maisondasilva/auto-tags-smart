Автоматично виявляє та призначає існуючі позначки для записів за допомогою інтелектуального аналізу контенту та розширених опцій фільтрації.

== Опис ==

Розумні авто-позначки — це сучасна система автоматичного позначення, яка інтелектуально виявляє та призначає існуючі позначки для ваших записів.

= Основні можливості =

* **Розумне виявлення**: Аналізує заголовки та вміст записів для пошуку відповідних позначок
* **Розширені налаштування**: Повний контроль над тим, як позначки виявляються та застосовуються
* **Фільтри по категоріях**: Автоматичне застосування позначок лише до вибраних категорій
* **Контроль довжини**: Встановлення мінімальної довжини позначки
* **Чутливість до регістру**: Можливість розрізняти верхній та нижній регістр
* **Сучасний інтерфейс**: Чиста та інтуїтивно зрозуміла панель адміністрування
* **Статистика**: Перегляд статистики блогу прямо у Майстерні

= Розширені можливості =

* Блокування користувацьких позначок (опціонально)
* Аналіз релевантності контенту
* Автоматичні пропозиції нових позначок на основі контенту
* Повне очищення при видаленні
* Сумісність з останніми хуками WordPress

= Як це працює =

1. Встановіть та активуйте плагін
2. Налаштуйте параметри на сторінці "Розумні Авто-позначки" у меню Записи
3. Виберіть, аналізувати заголовки, вміст або обидва варіанти
4. Налаштуйте фільтри по категоріях за потреби
5. Виберіть розширені налаштування на свій розсуд
6. Збережіть зміни та починайте створювати записи!

Плагін автоматично виявить існуючі позначки у вашому контенті та застосує їх до записів.

= Вимоги =

* WordPress 5.0 або новіший
* PHP 7.4 або новіший
* Мінімум пам'яті: 64MB

== Встановлення ==

1. Завантажте папку 'auto-tags' у директорію '/wp-content/plugins/'
2. Активуйте плагін через меню 'Плагіни' у WordPress
3. Перейдіть у "Записи > Розумні авто-позначки" для налаштування параметрів
4. Налаштуйте параметри на свій розсуд
5. Збережіть зміни та починайте використовувати!

= Ручне встановлення =

1. Завантажте zip-файл плагіна
2. Розпакуйте файли у '/wp-content/plugins/auto-tags'
3. Активуйте плагін в адмін-панелі
4. Налаштуйте параметри у Записи > Розумні авто-позначки

== Часті питання ==

= Чи працює плагін з уже існуючими записами? =

Так! Плагін працює при редагуванні та збереженні існуючих записів. Щоб застосувати позначки до всіх записів одразу, використовуйте функцію масового оновлення WordPress.

= Чи можу я вибрати, які категорії будуть охоплені? =

Так! Ви можете увімкнути фільтр по категоріях і вибрати, до яких категорій автоматично застосовувати позначки.

= Чи створює плагін нові позначки? =

Ні, плагін лише виявляє та застосовує вже існуючі позначки вашого блогу. Він не створює нові позначки автоматично.

= Як переглянути, які позначки були застосовані? =

Ви можете побачити застосовані позначки у редакторі записів, як і будь-які інші позначки. Плагін також веде статистику у панелі адміністрування.

= Чи впливає плагін на продуктивність сайту? =

Плагін оптимізований для продуктивності та працює лише при створенні/редагуванні записів. Плагін не впливає на швидкість завантаження сторінок для відвідувачів.

= Чи можу я тимчасово вимкнути плагін? =

Так! Ви можете зняти галочку "Увімкнути розумні авто-позначки" у налаштуваннях, щоб тимчасово вимкнути плагін без видалення.

= Чи втратяться налаштування при видаленні? =

Це залежить від налаштувань видалення. Якщо така опція увімкнена, всі налаштування буде видалено. Якщо ні — залишаться у базі даних.

== Примітки для розробників ==

Плагін розроблений з урахуванням найкращих практик WordPress:

* Використовує відповідні хуки та фільтри
* Добре документований та організований код
* Сумісний з останніми версіями WordPress
* Оптимізований по продуктивності
* Адаптивний інтерфейс
* Реалізовано безпеку (nonce, санітизація, валідація)

= Доступні хуки =

* `at_before_tagging` — виконується перед застосуванням позначок
* `at_after_tagging` — виконується після застосування позначок
* `at_tag_found` — виконується при виявленні позначки
* `at_excluded_tag` — виконується при виключенні позначки

= Доступні фільтри =

* `at_minimum_tag_length` — змінює мінімальну довжину позначки
* `at_content_to_analyze` — змінює аналізований контент
* `at_found_tags` — змінює знайдені позначки перед застосуванням
