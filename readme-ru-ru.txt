Автоматически обнаруживает и назначает существующие метки для записей с помощью интеллектуального анализа контента и расширенных опций фильтрации.

== Описание ==

Умные авто-метки — это продвинутая система автоматического тегирования, которая интеллектуально обнаруживает и назначает существующие теги для ваших записей.

= Основные возможности =

* **Умное обнаружение**: Анализирует заголовки и содержимое записей для поиска совпадающих меток
* **Расширенные настройки**: Полный контроль над тем, как теги обнаруживаются и применяются
* **Фильтры по рубрикам**: Автоматическое применение меток только к выбранным рубрикам
* **Контроль длины**: Установка минимальной длины меток
* **Чувствительность к регистру**: Возможность различать верхний и нижний регистр
* **Современный интерфейс**: Чистая и интуитивно понятная панель администрирования
* **Статистика**: Просмотр статистики блога прямо в консоли

= Расширенные возможности =

* Блокировка пользовательских меток (опционально)
* Анализ релевантности контента
* Автоматические предложения новых тегов на основе контента
* Полная очистка при удалении
* Совместимость с последними хуками WordPress

= Как это работает =

1. Установите и активируйте плагин
2. Настройте параметры на странице "Умные Авто-метки" в меню Записи
3. Выберите, анализировать ли заголовки, содержимое или оба варианта
4. Настройте фильтры по рубрикам при необходимости
5. Выберите расширенные настройки по своему усмотрению
6. Сохраните изменения и начните создавать записи!

Плагин автоматически обнаружит существующие метки в вашем контенте и применит их к записям.

= Требования =

* WordPress 5.0 или новее
* PHP 7.4 или новее
* Минимум памяти: 64MB

== Установка ==

1. Загрузите папку 'auto-tags' в директорию '/wp-content/plugins/'
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Перейдите в "Записи > Умные авто-метки" для настройки параметров
4. Настройте параметры по своему усмотрению
5. Сохраните изменения и начинайте использовать!

= Ручная установка =

1. Скачайте zip-файл плагина
2. Распакуйте файлы в '/wp-content/plugins/auto-tags'
3. Активируйте плагин в админ-панели
4. Настройте параметры в Записи > Умные авто-метки

== Часто задаваемые вопросы ==

= Работает ли плагин с уже существующими записями? =

Да! Плагин работает при редактировании и сохранении существующих записей. Чтобы применить метки ко всем записям сразу, используйте функцию массового обновления WordPress.

= Могу ли я выбрать, какие рубрики будут затронуты? =

Да! Вы можете включить фильтр по рубрикам и выбрать, к каким рубрикам автоматически применять метки.

= Плагин создает новые метки? =

Нет, плагин только обнаруживает и применяет уже существующие метки вашего блога. Он не создает новые метки автоматически.

= Как посмотреть, какие метки были применены? =

Вы можете увидеть примененные метки в редакторе записей, как и любые другие метки. Плагин также ведет статистику в панели администрирования.

= Влияет ли плагин на производительность сайта? =

Плагин оптимизирован для производительности и работает только при создании/редактировании записей. Плагин не влияет на скорость загрузки страниц для посетителей.

= Могу ли я временно отключить плагин? =

Да! Вы можете снять галочку "Включить умные авто-метки" в настройках, чтобы временно отключить плагин без удаления.

= Потеряются ли настройки при удалении? =

Это зависит от настроек удаления. Если такая опция включена, все настройки будут удалены. Если нет — останутся в базе данных.

= Доступные хуки =

* `at_before_tagging` — выполняется перед применением меток
* `at_after_tagging` — выполняется после применения меток
* `at_tag_found` — выполняется при обнаружении метки
* `at_excluded_tag` — выполняется при исключении метки

= Доступные фильтры =

* `at_minimum_tag_length` — изменяет минимальную длину метки
* `at_content_to_analyze` — изменяет анализируемый контент
* `at_found_tags` — изменяет найденные метки перед применением
